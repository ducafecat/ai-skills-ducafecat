# Flutter 项目初始化脚本

使用 ducafe_ui_core + GetX 初始化规范的 Flutter 项目结构。

## 使用方式

在 Flutter 项目根目录下执行：

```bash
node /path/to/index.js
```

或全局安装后使用：

```bash
npm link
flutter-getx-init
```

## 功能

- 安装 get 和 ducafe_ui_core 依赖
- 创建 lib/pages/index 业务页面
- 创建 lib/global.dart 全局模块
- 创建 lib/common/ 通用模块目录结构
- 重写 lib/main.dart
